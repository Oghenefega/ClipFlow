Recordings cards — v4 · selection rethink

Option A (live below): the left checkbox is gone. Click a card to select it — it lights up purple. Done stays the green ✓ on the right. One checkmark, color-coded states, more room for the name. Try selecting a few and watch the footer count.

Does selection read clearly without a box? That's the call. Purple card = selected / armed for generation (footer count updates). Green card + ✓ = done. Plain = idle. Click a card to select; click a green ✓ → red ✕ → remove from done. Hover a name for the full filename + size. Top-right still toggles AR ⇄ |.

Alternative B — keep a checkbox, but it means selection only (never doubles as done)

With B, a done card shows an empty box + the green ✓ (box never auto-checks for done), so the two only coexist when a card is both selected and done — and they look different (purple box vs bare green ✓). Safer/more familiar, but keeps the box (~22px) and a second checkmark in that rare overlap.
